# Financial Data Analysis Tool

This Python tool allows you to analyze financial data, generate reports, and visualize stock prices using popular libraries like Pandas, NumPy, and Matplotlib.

## Features

- Load financial data from CSV files.
- Calculate basic statistics and generate summary reports.
- Visualize stock prices over time with line charts.
- Calculate and plot moving averages to identify trends.

Usage
Place your financial data CSV file in the project directory.

Modify the filename and stock_symbol variables in the main script with your data file's name and stock symbol.

Run the script:

python financial_data_analysis.py

Stock Price Over Time: Line chart showing stock prices.
